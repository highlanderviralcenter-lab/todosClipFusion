⚡ KERNEL & GPU (CORRIGIDO)

Nada de NVIDIA.

Em /etc/default/grub:

GRUB_CMDLINE_LINUX_DEFAULT="quiet mitigations=off i915.enable_guc=2 i915.modeset=1 nouveau.blacklist=1"


Depois:

update-grub
reboot

Driver Intel

Teste:

vainfo | grep iHD


Se aparecer iHD → sucesso.

🧠 MEMÓRIA
ZRAM
apt install -y zram-tools


Em /etc/default/zramswap:

PERCENT=50
ALGO=zstd


Restart:

systemctl restart zramswap

Swap SSD
fallocate -l 4G /swapfile
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
echo '/swapfile none swap sw 0 0' >> /etc/fstab

🧠 IA (WHISPER CORRIGIDO)

Nada de CUDA.

pip install faster-whisper


Uso correto:

from faster_whisper import WhisperModel

model = WhisperModel("medium", device="cpu", compute_type="int8")

Modelos ficam em:

/home/$USER/models

🎬 FFmpeg VA-API
apt install -y ffmpeg


Pipeline:

ffmpeg -hwaccel vaapi -hwaccel_device /dev/dri/renderD128 \
-vaapi_driver iHD -i input.mp4 \
-c:v h264_vaapi -preset fast -qp 23 \
-c:a copy output.mp4


Intel HD 520 faz 60-80fps tranquilo.
