🔧 Ponte / Host Bridge
Intel Host Bridge/DRAM Registers  
• Função: conecta CPU ↔ DRAM ↔ Chipset.
• Classe PCI: 0x06 (Pontes), Subclasse Host Bridge 0x00.
• Essencial para inicialização de memória e comunicação com controladores.
• No seu caso: Xeon E3‑1200 v5/E3‑1500 v5/6th Gen Core Processor Host Bridge (Skylake).
